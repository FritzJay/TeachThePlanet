# TeacherHub
One stop shop for your favorite teachers

This repository is for:
- Documentation
- Communication
- Framework library

[Rest API](https://github.com/FritzJay/TeacherHub.API)

[Fact Fluency](https://github.com/FritzJay/TeacherHub.FactFluency)


Student:
- Sign Up
- Login
- Logout
- Join a class
- Practice
- Testing
- At home test params
- At school test params
- Custom test params

Teacher:
- Sign Up
- Login
- Logout
- Create a class
- Invite student
- Class level test params
- Remove students from class
- View reports
- Invite parent

Parent:
- Sign Up
- Login
- Sign Up a student
- Set test params
- View reports
- Request invitation
